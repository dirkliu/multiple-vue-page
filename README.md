# multiple-vue-page

多页vue应用demo项目

## 需要安装的工具
```
npm install webpack -g
npm install webpack-dev-server
```
## node依赖安装
```
npm install
```

## 开始运行
```
npm run dev
```

## 构建
```
npm run build
```
